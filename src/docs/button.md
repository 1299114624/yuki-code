## Button 按钮
常用的操作按钮, 在默认类型按钮上加类`yuki-button`即可.
&emsp;&emsp;

### 基础用法
:::demo

```html
<el-button class='yuki-button'>普通按钮</el-button>
<el-button class='yuki-button enhance'>强调按钮</el-button>
<yuki-button/>
```


:::
